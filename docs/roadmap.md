# Roadmap de Evolução

## Curto Prazo
- [ ] Categorias gerenciáveis via UI (CRUD de categorias)
- [ ] Orçamentos: CRUD e cálculo de progresso por categoria
- [ ] Notificações: regras e disparos (ex.: ultrapassar orçamento)
- [ ] Melhorar UX: componentes reutilizáveis, feedbacks e validações

## Médio Prazo
- [ ] Filtros avançados e busca em transações
- [ ] Gráficos mais ricos (ex.: charts interativos)
- [ ] Exportação avançada: XLSX, formatos customizados
- [ ] Internacionalização (i18n)
- [ ] Testes automatizados (unit/integration) frontend e backend

## Longo Prazo
- [ ] Multi-usuário com times e compartilhamento
- [ ] Integração bancária (importação automática)
- [ ] Assinaturas e planos (limites e features)
- [ ] Observabilidade (APM, tracing)
- [ ] Deploy em cloud (CI/CD)
